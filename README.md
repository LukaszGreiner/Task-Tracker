A lightweight Python command-line app to manage tasks with persistent storage, status workflow, and timestamp tracking.

## Highlights

- Add, update, delete tasks
- Track progress: `todo` → `in-progress` → `done`
- Persist data in `tasks.json`
- UTC timestamps in storage, local-time display in output

## Quick Start

1. Open a terminal in this folder.
2. Run commands with Python:

```bash
python task-cli.py --help
```
